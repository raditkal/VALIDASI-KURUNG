Validasi Tanda Kurung

Web native untuk mengecek keseimbangan (), [], dan {} pada kalimat, ekspresi perhitungan, atau potongan kode.

Hasil validasi akan muncul di sini.

Konsep Bahasa & Otomata

Program ini menggunakan model stack seperti Pushdown Automata (PDA):

  • Setiap tanda kurung buka disimpan (push) ke stack.
  • Setiap tanda kurung tutup harus cocok dengan elemen teratas (pop).
  • Input valid jika tidak ada mismatch dan stack kosong di akhir.
